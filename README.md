This repository documents my understanding of essential networking concepts that form the foundation of cybersecurity. It covers network communication, addressing, protocols, traffic management, and access control mechanisms that are commonly encountered in security operations, penetration testing, and network administration.

---

## Topics Covered

### 1. OSI Model
- The seven layers of the OSI model
- Functions of each layer
- Real-world examples of protocols and devices at each layer
- Troubleshooting using the OSI model

### 2. TCP/IP Model
- The four layers of the TCP/IP model
- Comparison between the OSI and TCP/IP models
- Common protocols used within each layer

### 3. IP Addressing
- IPv4 addressing
- Public vs Private IP addresses
- Static vs Dynamic IP addresses
- Network and Host portions of an IP address

### 4. Subnetting
- CIDR notation
- Subnet masks
- Network address
- Broadcast address
- Host address calculations
- Practical subnetting exercises

### 5. MAC Addressing
- What a MAC address is
- MAC address structure
- Communication within a Local Area Network (LAN)
- ARP (Address Resolution Protocol)

### 6. Domain Name System (DNS)
- DNS resolution process
- Common DNS record types
- Importance of DNS in network communication

### 7. Dynamic Host Configuration Protocol (DHCP)
- Purpose of DHCP
- The DORA process:
  - Discover
  - Offer
  - Request
  - Acknowledge

### 8. TCP vs UDP
- Connection-oriented vs Connectionless communication
- Reliability and speed
- Common protocols that use TCP and UDP
- Practical use cases

### 9. Network Topologies
- Star
- Bus
- Ring
- Mesh
- Tree
- Hybrid

### 10. Firewalls
- Purpose of firewalls
- Packet filtering
- Stateful inspection
  

### 11. Access Control Lists (ACLs)
- Standard ACLs
- Extended ACLs
- Traffic filtering
- Permit and deny statements
- Practical ACL examples

### 12. Implicit Deny vs Explicit Deny
- Understanding implicit deny
- Understanding explicit deny
- Importance of rule order in ACLs
- Security best practices
